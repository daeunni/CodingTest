## 재귀함수 (Recursive Function) 

- **스택** 자료구조의 한 형태라는걸 기억할 것 
> 마지막 호출 함수가 먼저 수행을 끝내야, 그 앞의 함수 호출이 종료되기 때문임!

- 항상 꼭 **종료조건**을 명시해야 함 (if 문 등으로) 

```python 
def fact(n):
  if n <= 1 :       # 팩토리얼은 양수에 대해서만 성립되므로 이 조건이 종료조건으로 작용한다. 
    return 1 
  
  return n * fact(n-1)    # 수학적 점화식형태로 작성 

```
